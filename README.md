# Структура AST 
## Типы узлов:
- 1 Узел AstNode

  Базовый узел, из которого наследуются все последующие.

  Атрибуты:
  type - тип данных;
  name - идентификатор.

- 2 ComplexDeclarationNode

  Узел, представляющий объявление переменной комплексного числа
  
  Атрибуты:
    - Name – идентификатор
    - Type – тип значения
    - ValueRealPart – значение реальной части
    - ValueImagePart – значение мнимой части

 - 3 DoubleLiteralNode

  Узел, представляющий числовое значение 
  
  Атрибуты:
    - Name – идентификатор
    - Type – тип значения
    - Value – числовое значение
